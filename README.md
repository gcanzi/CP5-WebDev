# 🍔 GourmetOn - Landing Page

![React](https://img.shields.io/badge/React-19.2.5-blue?logo=react)
![Vite](https://img.shields.io/badge/Vite-8.0.10-646CFF?logo=vite)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4.2.4-38B2AC?logo=tailwind-css)

Landing Page moderna e responsiva para o aplicativo de delivery de comida **GourmetOn**, desenvolvida com React, Vite e Tailwind CSS v4.

## 📋 Sobre o Projeto

O GourmetOn é um aplicativo de delivery de comida que conecta usuários a centenas de restaurantes, oferecendo entrega rápida, variedade de opções e pagamento facilitado. Esta landing page foi desenvolvida para apresentar os principais benefícios e funcionalidades do aplicativo com um design moderno e sofisticado.

## ✨ Funcionalidades

- **Menu de Navegação Fixo**: Header responsivo com efeito glassmorphism e smooth scroll
- **Hero Section Impactante**: Apresentação com gradientes, estatísticas e CTAs
- **Seção de Benefícios**: 3 cards com animações hover e ícones personalizados
- **Doces em Destaque**: Galeria de 5 sobremesas com nomes em português
- **Depoimentos**: 3 avaliações de clientes com sistema de estrelas
- **Formulário de Contato**: Captura de e-mails para newsletter
- **Footer Completo**: Informações de contato, links e créditos da equipe
- **Design Responsivo**: Mobile First, 100% adaptável a todos os tamanhos de tela

## 🚀 Tecnologias Utilizadas

- **React 19.2.5** - Biblioteca JavaScript para construção de interfaces
- **Vite 8.0.10** - Build tool moderna e extremamente rápida
- **Tailwind CSS 4.2.4** - Framework CSS utility-first com plugin oficial para Vite
- **JavaScript ES6+** - Recursos modernos do JavaScript

## 🎨 Design Moderno

- **Gradientes Vibrantes**: Botões e backgrounds com gradientes laranja
- **Glassmorphism**: Header com backdrop blur
- **Animações Suaves**: Hover effects, scale e transições
- **Blobs Animados**: Efeitos de blur no background
- **Sombras Dinâmicas**: Cards com sombras que mudam no hover
- **Tipografia Bold**: Uso de font-black para títulos impactantes

## 🎯 Seções da Landing Page

## 📱 Responsividade

O design foi desenvolvido seguindo a metodologia **Mobile First**, utilizando breakpoints do Tailwind CSS:

- **Mobile**: < 768px (1 coluna)
- **Tablet**: 768px - 1023px (2 colunas)
- **Desktop**: 1024px+ (3-5 colunas)
